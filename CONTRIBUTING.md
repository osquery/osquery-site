# Contributing to osquery-site

We want to make contributing to this project as easy and transparent
as possible.

## Pull Requests

We actively welcome your pull requests.

1. Fork the repo and create your branch from `source`.
2. Ensure the site runs locally.
   1. Instructions available on the [README.md](README.md).
3. Make sure your code lints.
4. If you haven't already, complete the Contributor License Agreement
   ("CLA").

## Contributor License Agreement ("CLA")

You must submit a Contributor License Agreement (CLA) before we can
accept any of your pull requests. You only need to submit one CLA for
any of osquery's open source projects.

This is managed through the Linux Foundations's EasyCLA. It will
comment appropriately on your PR.

## Issues

We use GitHub issues to track public bugs. Please ensure your
description is clear and has sufficient instructions to be able to
reproduce the issue.
